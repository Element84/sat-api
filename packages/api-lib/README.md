## @sat-utils/api-lib

### Unit Tests
```
$ yarn
$ yarn test
```

### Integration Tests
Navigate to the integration directory
```
$ cd ./tests/integration
```
Use the environment variable `DOCKER_NAME` to set your Docker host name.
Normally `localhost`.
```
$ export DOCKER_NAME=localhost
```
To run the tests
```
$ .tests/integration/runIntegration.sh
```
